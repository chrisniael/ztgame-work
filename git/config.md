# 配置

```shell
git config --global user.name your_name    # 将 your_name 改成你自己的名字
git config --global user.email your_email@example.com    # 将 your_email@example.com 改成你自己的邮件
git config --global core.editor vim
git config --global color.branch auto
git config --global color.decorate auto
git config --global color.diff auto
git config --global color.grep auto
git config --global color.interactive auto
git config --global color.pager auto
git config --global color.showbranch auto
git config --global color.status auto
git config --global color.ui auto
```
