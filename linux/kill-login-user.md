# 踢出登陆的用户

## 查看在线用户

```shell
who
```

```
root     pts/0        2016-08-27 13:18 (222.72.79.198)
shenyu   pts/1        2016-08-27 00:06 (222.72.79.198)
```

## 确认自己是哪个

```shell
who am i
```

```
root     pts/0        2016-08-27 13:18 (222.72.79.198)
```

## 提出用户

这里提出 `shenyu` 这个用户

```shell
pkill -KILL -t pts/1
```

注意权限的问题，`root` 用户可以踢出任何用户，而其他用户只能踢出自己的账户。
